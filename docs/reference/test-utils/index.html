Variables

VariableDescription
TestApiProviderThe TestApiProvider is a Utility API context provider that is particularly well suited for development and test environments such as unit tests, storybooks, and isolated plugin development setups.It lets you provide any number of API implementations, without necessarily having to fully implement each of the APIs.A migration from ApiRegistry and ApiProvider might look like this, from:
 
renderInTestApp(
  <ApiProvider
    apis={ApiRegistry.from([
      [identityApiRef, mockIdentityApi as unknown as IdentityApi]
    ])}
  >
    {...}
  </ApiProvider>
)

To the following:

renderInTestApp(
  <TestApiProvider apis={[[identityApiRef, mockIdentityApi]]}>
    {...}
  </TestApiProvider>
)

Note that the cast to IdentityApi is no longer needed as long as the mock API implements a subset of the IdentityApi. |
